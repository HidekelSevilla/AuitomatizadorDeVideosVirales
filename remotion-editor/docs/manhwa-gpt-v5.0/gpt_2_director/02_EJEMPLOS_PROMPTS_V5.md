# Ejemplos de prompts V5

Los ejemplos demuestran mecánica, no contenido para copiar.

## 1. Base de personaje

```text
Character identity reference sheet of [NAME]. Exactly one Korean [age/role], [build], [face/hair/eyes], neutral relaxed expression. Full body from hair to soles, orthographic front eye-level view, minimal perspective, both open empty hands visible beside the hips, both feet visible. Clean dry [simple base clothing]. Even soft studio illumination, seamless medium-gray background, crisp inked Korean manhwa character design, controlled 2D cel shading.
```

No añade lluvia, pose heroica, arma, poder, rim light ni escenario.
